Struct opendal::FuturesIoAsyncReader

source · 
pub struct FuturesIoAsyncReader { /* private fields */ }
Expand description

FuturesAsyncReader is the adapter of [AsyncRead], [AsyncBufRead] and [AsyncSeek] for Reader.

Users can use this adapter in cases where they need to use [AsyncRead] related trait.

FuturesAsyncReader also implements Unpin, Send and Sync
